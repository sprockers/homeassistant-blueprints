# Home Assistant Blueprints

Here you can find Home Assistant blueprints created by Chris Wood.

I really enjoy tinkering with Home Assistant, but after I had few hundred automations under my belt
I realised that reusing the same automation across 10 different rooms was a faff which required a lot
of manual entity copy/pasting which was prone to error.

The solution is to create a blueprint. It's definitely a step up from creating an automation, but
I think it's worth the effort!
